Detailed Description

UART multicon interface of the FM4, which can be used to connect the BLE module to the FM4. You can test the BLE module duplex connection with this code: 

#include "init.h"
#include "uart_multicon.h"
int main(){
 initBoard();
 cppp_initUart3();
 cppp_uart3DuplexTest();
 return 0;
}
