Detailed Description

You can use board_test.h to print all the periphery of the cppp board on the lc display. Just create the following code inside the main.c. 

#include "init.h"
#include "board_test.h"
int main(){
 initBoard();
 cppp_initBoardTest();
 cppp_boardTest();
 return 0;
}
